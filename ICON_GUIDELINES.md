### Xygones – Icon Requirements

-------------------------------------------------



Required Icon Sizes:

• 16 × 16 px  – Extension toolbar icon (default small icon)

• 48 × 48 px  – Extension management page icon

• 128 × 128 px – Chrome Web Store listing icon



File Naming:

Please use the same file names so the extension loads correctly:

\- icon16.png

\- icon48.png

\- icon128.png
